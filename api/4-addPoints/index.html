  • 参数：

    • {Object} point 标准空间要素

      • {Object} attributes 属性信息，用于气泡展示等其他要素

        • {String} id || ID id必传

      • {ol.geom.Geometry} 空间位置信息

      • {String} 要素类型

    • {Object} options 相关参数

      • {String} layerName 图层name
